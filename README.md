SCPCraft is an SCP mod that adds three SCPs to minecraft: [SCP-005](https://scp-wiki.wikidot.com/scp-005), [SCP-009](https://scp-wiki.wikidot.com/scp-009), [SCP-173](https://scp-wiki.wikidot.com/scp-173)

SCP-005 is a key that opens iron doors when right-clicked on them. It is crafted with three copper ingots in a forward slash like diagonal line.
SCP-009 is a block of red ice that spreads itself to any block that is associated with water; such as ice, snow, and water sources. It is crafting by surrounding a red dye with packed ice.
SCP-173 is a naturally-spawning entity that chases nearby players. The only way to stop it is by looking at it, which prevents it from moving. SCP-173 is invulnerable to all sources of damage, except for pickaxes.
